Como Usar o Sistema

Este sistema foi projetado para auxiliar na gestão de pacientes que realizam raio-x torácico. Abaixo estão algumas instruções básicas para usar a plataforma:

  • Atualizar Lista: Clique no botão "Atualizar Lista" para carregar novos dados.
  • Pacientes Prioritários: Verifique a lista de alta prioridade no painel esquerdo.
  • Pacientes de Menor Urgência: Consulte o painel direito para casos menos críticos.

Dúvidas Frequentes

  • Como faço para adicionar novos pacientes? Atualmente, os raio-x que entram na pasta "dicom", aparecem na página assim que o botão "atualizar lista" é pressionado.
  • Onde encontro os resultados analisados? Acesse a seção "Pacientes Analisados".
  • Como entrar em contato? Envie um e-mail para contato@exemplo.com.
